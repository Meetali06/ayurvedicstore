# 🌿 Ayurly — Ayurvedic Store
> A beautiful, static front-end website for **Ayurly**, your one-stop destination for authentic Ayurvedic products, herbal remedies, and wellness essentials.
---
## 📖 About Ayurly
**Ayurly** is an online Ayurvedic store that brings the ancient wisdom of Ayurveda right to your doorstep. From herbal supplements and organic skincare to traditional oils and wellness teas, Ayurly offers a curated collection of natural products sourced from trusted, authentic origins.
This project is a **static website** built as part of a summer internship to practice and strengthen front-end web development skills using HTML, CSS, and JavaScript.
---
## ✨ Features
- 🛍️ **Product Showcase** — Browse a wide range of Ayurvedic products with descriptions and images
- 🌱 **Category Navigation** — Easily explore categories such as Skincare, Hair Care, Supplements, Oils, and Teas
- 🏠 **Home Page** — Welcoming landing page with hero banner, featured products, and promotions
- 📦 **Product Detail View** — Detailed information about individual products including ingredients and benefits
- 📬 **Contact Section** — Get in touch form or contact details for customer inquiries
- 📱 **Responsive Design** — Optimized for desktop, tablet, and mobile devices
- 🎨 **Clean & Natural UI** — Earthy color palette and calm design inspired by Ayurvedic aesthetics

## 🚀 Getting Started
Since this is a static website, no installation or build process is needed.
### Run Locally
1. **Clone the repository**
   ```bash
   git clone https://github.com/Meetali06/ayurvedicstore.git
   ```
2. **Navigate into the project folder**
   ```bash
   cd ayurvedicstore
   ```
3. **Open in your browser**
   - Simply open `index.html` in any modern web browser, **or**
   - Use a local server extension like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code for a better development experience.
